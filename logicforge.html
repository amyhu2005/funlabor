The IKEA Manual for Code

Logic Forge

OVERVIEW

Logic Forge is a project-based coding tutor designed for people who want to learn by building things they actually care about. I made this because I was vibecoding a lot, and while it was fun, it was not intellectually stimulating enough. I wanted AI to help me code without taking away the most interesting part: figuring out the logic behind how the project actually works. At the same time, learning to code from scratch can feel overwhelming because you have to search through endless tutorials, documentation, and random answers online just to figure out what to do next. Logic Forge turns your project idea into a guided, step-by-step building process, almost like an IKEA instruction manual for coding. The goal is not to let AI build everything for you. Instead, it leaves the interesting logic questions for you to figure out, while AI helps with execution, syntax, structure, and explanations. It is built for people who want the fastest path to actually understanding code while still making projects that feel fun, personal, and useful.

01 // UPLOAD_YOUR_PROJECT_IDEA

Start by uploading or describing a project you want to build. Instead of learning from random practice problems, you learn coding through something you are actually excited to make.

02 // CHOOSE_AI_HANDLING

You can decide whether you want to build both the frontend and backend yourself, or let the AI handle parts like the frontend so you can focus more deeply on the backend logic.

03 // GUIDED_PLANNING_WORKSHEET

Before you start coding, Logic Forge helps you break your idea into mini goals and an implementation plan. If you get stuck, you can view a recommended plan and use it as a guide. At the end of the planning stage, you can choose what language or tools you want to use. The app can also recommend the best option based on your project.

04 // EDITABLE_PROJECT_DOCUMENTS

Your original idea and implementation plan are saved as Markdown files on your dashboard. You can edit them anytime, and the AI teacher uses these documents to guide the learning process.

05 // IKEA_STYLE_INSTRUCTION_MANUAL

Once your plan is ready, Logic Forge generates a step-by-step instruction manual for building your app. It does not just give you the answer; it leaves important logic puzzles for you to solve so you actually learn.

06 // CONCEPT_RECOMMENDATIONS

As you build, the app recommends coding concepts you should learn or read about, based on what your project requires.

07 // BUILT_IN_AI_TEACHER

A full chatbot is integrated into the process, so you can ask questions, get unstuck, and understand why certain code decisions are being made.

08 // INTEGRATED_CLI_SYSTEM

Logic Forge is designed to connect with a command-line workflow, helping you move from planning to actually building your project in a more structured way.

LEARN_THE_FUN_PARTS

The main goal is to make coding feel less like memorizing syntax and more like solving interesting design and logic problems. AI helps with the boring or confusing logistics, while you focus on understanding and creating.
